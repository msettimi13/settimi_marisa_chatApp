# Be Kind Chat and Listen

## About

A chat app that can be used for people that need someone to talk to and for others to help and listen.

View the live link on heroku [here](https://settimi-m-chat-app.herokuapp.com/).

## Built with

1. HTML5
2. SASS/CSS
3. JavaScript
4. Vue.js
5. Node.js
6. Socket.io
